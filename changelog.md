# Changelog

All notable changes to Grimoire will be documented in this file.

## 2026-03-21

### Added
- Tarot module: card library, reading flow, spread builder, journal, synthesis engine
- Astrology module: birth chart creator, placements, aspects, chart interpretation, horoscopes
- Herbology module: herb library with search/filter, herb detail views, blend builder
- Crystal module: crystal library, crystal detail views, collection tracking, intention-based pairing
- Dream journal module: dream entry form, journal list, symbol library, pattern dashboard
- Moon phase module: moon calendar, today's moon phase, ritual suggestions, moon journal
- Numerology module: numerology calculator, number cards, profile views
- Admin dashboard with dynamic table editor for all Supabase tables
- Auth via Supabase magic link
- Navbar and footer with module navigation
- Full design system (STYLE.md) with earthy/witchy aesthetic
- Static data files for all modules (tarot, astrology, crystals, herbs, dreams, moon rituals, numerology)
- Spread templates with visual canvas positioning
- Synthesis engine for template-based tarot interpretations
- Dark/light theme support via ThemeProvider
